# Car Racing Game WinForms

Простая гоночная игра на C# Windows Forms.

## Функциональность

### 🏎️ Управление
- Красная машинка (pictureBox1): WASD
- Синяя машинка (mashinka2): стрелки
- Ограничение движения по горизонтали

### 🪙 Монеты и счет
- Сбор монет увеличивает счет
- Красная машинка собирает coin1
- Синяя машинка собирает coin2
- Счет отображается в label1 и label2

### 👵 Препятствия (бабки)
- Столкновение с препятствием уменьшает счет
- Препятствия появляются случайно сверху
- babka_porshe для красной машинки
- babka_ferrari для синей машинки

### 🏁 Победа
- Для победы нужно набрать 30 очков
- Побеждает тот, кто первый достигнет 30
- Сообщение о победителе и закрытие игры

### 🛣️ Дорога
- Движущиеся полосы дороги (Doroga и Doroga2)
- Создание эффекта движения

## Интерфейс

- pictureBox1 — красная машинка (управление WASD)
- mashinka2 — синяя машинка (управление стрелками)
- coin1 и coin2 — монеты
- babka_porshe и babka_ferrari — препятствия
- Doroga и Doroga2 — полосы дороги
- label1 — счет красной машинки
- label2 — счет синей машинки
- timer1 — основной игровой таймер

## Куда можно встроить

- В игровые проекты для обучения
- В демонстрацию работы с таймером и коллизиями
- В примеры обработки клавиатурного ввода

## Требования

- Windows
- .NET Framework / .NET Core с WinForms
- Изображения для машинок, монет и препятствий

## Примечание

Игра использует столкновения через Bounds.IntersectsWith. Для корректной работы все pictureBox должны быть непрозрачными и иметь достаточный размер хитбокса.
